# Office Directory

# Application Does:
* This is a sortable directory of all the characters on the hit tv show "The Office". Ability to sort characters by name. 

# Technologies used:
* React
* CSS , Bootstrap
* HTML
* JavaScript


# Features
* React Cards For Each Employee
* Sortable Employees By Name
